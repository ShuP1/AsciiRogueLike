# Ascii Rogue Like

GCC is a console based minimal rogue like game. 

### Installing

Download last version

Linux
```
sudo apt-get install mono-runtime
```

## Running

* AsciiRogueLike.exe

Linux
```
mono AsciiRogueLike.exe
```

## Test With

* Windows 10 x64
* XUbuntu 16.04
